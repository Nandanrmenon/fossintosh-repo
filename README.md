# Fossintosh Repository
This repository is solely to maintain the list of the free and open source application. This repo is also used to fetch the list onto the Fossintosh app. 

## How to request app?
Very simple.
Just create a pull request with changes onto ```apps.json```.

For example
```
    {
        "id": "APP_ID",
        "name": "APP_NAME",
        "description": "This is an Foss App",
        "version": "APP_VERSION",
        "category": "APP_CATEGORY",
        "icon": "https://example.com/icon.png",
        "downloadUrl": "https://example.com/download.dmg",
        "homepage": "https://example.com/",
        "license": "APP_LICENSE",
        "author": "AUTHORS_NAME",
        "screenshots": [
            "https://example.com/screenshot1.png",
            "https://example.com/screenshot2.png"
        ]
    },
```
